Projet BUT S3

Application Recherche de Restaurants

Cartographie interactive des restaurants à proximité

Application web développée en JavaScript avec Webpack permettant de trouver les restaurants autour d'une adresse donnée. L'utilisateur saisit n'importe quelle adresse française et l'application affiche une carte interactive Leaflet avec tous les restaurants dans un rayon de 2 km.

Le projet suit une architecture orientée objet stricte avec trois classes : Restaurant, APIService et ApplicationRestaurants. Les données sont récupérées en temps réel via deux APIs publiques.

  • Carte interactive avec marqueurs Leaflet
  • Géocodage via l'API Adresse (data.gouv.fr)
  • Données restaurants via Overpass API (OSM)
  • Affichage : nom, cuisine, horaires, adresse
  • Interface responsive (desktop & mobile)

Projet BUT S3/4

FoxCorrector

Outil de correction automatisée de scripts Bash par IA

FoxCorrector est une application de bureau développée en Python permettant aux enseignants de corriger automatiquement des scripts Bash soumis par leurs étudiants. Elle exécute chaque script dans un environnement Docker isolé et sécurisé, puis analyse le résultat selon des critères personnalisables.

L'application dispose d'une interface graphique complète avec authentification utilisateur, tableau de bord de statistiques, configuration des exercices (mots-clés attendus, structures de code), détection de plagiat, et export des résultats en PDF/CSV.

UI (customtkinter) → Business Logic → Docker Sandbox
  • Exécution isolée dans Docker (mémoire + CPU limités)
  • Analyse par mots-clés & structures de code
  • Détection automatique de plagiat
  • Export des résultats en PDF / CSV
  • Multi-utilisateurs avec historique

Projet BUT S4

LoL Typing Test

Jeu de dactylographie sur le thème League of Legends

Application web de typing test développée en React avec Vite, inspirée de l'univers de League of Legends. Le joueur doit retranscrire un texte tiré du lore de Yunara aussi vite que possible en 60 secondes, avec un retour visuel en temps réel caractère par caractère.

Le projet repose sur un pattern de state management avancé via useReducer et le Context API de React, séparant proprement la logique métier de l'affichage en plusieurs composants spécialisés.

  • Coloration lettre par lettre en temps réel
  • Timer 60s avec alerte visuelle à 20s et 10s
  • Calcul du WPM à la fin de chaque partie
  • Détection mots corrects / incorrects (soulignement)
  • Historique des parties dans le state global

Application Mobile — Le Petit Prince

Application React Native / Expo pour une école primaire

Application mobile développée en React Native avec Expo Router pour l'école "Le Petit Prince". Elle permet aux parents et enseignants de consulter les actualités de l'école, les galeries photos, et le calendrier des événements à venir.

Les données sont récupérées en temps réel via une API REST hébergée sur un serveur externe. L'application gère la persistance locale avec AsyncStorage pour les favoris et la mémorisation des identifiants.

  • Authentification avec option "Se souvenir de moi"
  • Articles avec recherche, photos et mise en favoris
  • Galeries photos dépliables avec légendes
  • Calendrier des événements à venir
  • Favoris persistants par catégorie (AsyncStorage)
  • Support thème clair / sombre automatique
